Experience

Belum ada keterangan lebih lanjut ...

Di BETIS Fasilkom UI 2022, saya mencoba untuk berpindah divisi, yaitu Web Development. Di divisi tersebut saya berfokus pada sisi front-end dan menggunakan framework ReactJS. Tugas saya di divisi tersebut adalah membuat komponen Try Out Multiple Choice dan saya juga membuat komponen Profile dari pengguna akun tersebut. Untuk melihat bagian code yang telah saya implementasikan bisa dilihat pada repository aurora di dalam folder components.

DI PMB Fasilkom UI 2021, saya diterima sebagai Mentor. Tugas seorang mentor ialah membantu mahasiswa baru untuk beradaptasi dengan lingkungan dan budaya yang ada pada Universitas Indonesia. Selain itu, mentor juga memberikan tips and tricks agar bisa survive di Fasilkom UI serta memberikan pengalaman dan manfaat yang akan didapatkan jika lebih aktif di organisasi atau kepanitiaan yang ada di Fasilkom UI.

Di BETIS Fasilkom UI 2021, saya diterima di divisi Akademis dan Pengajar. Di divisi tersebut, saya bertugas sebagai pengajar kimia untuk siswa SMA kelas 12 maupun alumni yang akan berjuang menghadapi ujian masuk Perguruan Tinggi Negeri (SBMPTN atau SIMAK UI).
